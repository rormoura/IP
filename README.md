# Repositório de Algoritmos em C

Este repositório armazena programas desenvolvidos na linguagem C para a resolução de problemas lógicos, análise de matrizes e manipulação de arquivos.

## Estrutura do Repositório

| Arquivo | Descrição |
| --- | --- |
| `sudoku.c` | Programa que lê uma matriz de um arquivo de texto e verifica a validade da solução de um jogo de Sudoku, analisando as submatrizes e a matriz global. |
| `primo.c` | Algoritmo que recebe um número inteiro e verifica se ele é um número primo por meio de divisões sucessivas. |
| `cacaPalavras.c` | Código que lê uma matriz de caracteres e uma lista de palavras de um arquivo de texto, busca as palavras em oito direções diferentes e salva os resultados em um arquivo binário. |
| `ListadeEncomendas.c` | Sistema de gerenciamento que lê dados de encomendas de um arquivo, permite identificar entregas realizadas para remoção do vetor e atualiza o arquivo original. |
